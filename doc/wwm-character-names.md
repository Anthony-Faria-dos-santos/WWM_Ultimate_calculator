## Notes Culturelles

### Naming Conventions
- **Bird Names (鳥名)**: Mo Mountain Tao elders use bird code names (Peng, He, Lu, Yan, etc.)
- **Title-Based**: Many NPCs known by titles (River Earl, Hall Master, East Tower Prince)
- **Double Names**: Characters often have 2-3 names (birth name, courtesy name, code name)

### Key Relationships
- **江晏 (Jiang Yan)** = "River calm" → **江無浪 (Jiang Wulang)** = "River without waves" (same meaning, alias)
- **紅線 (Hong Xian)** = "Red thread" symbolizes destiny/bonds
- **洛神 (Luo Shen)** = Reference to River Goddess, Han's disguise identity
- **東闕公子 (East Tower Prince)** = Wen Wuque's male persona

### Historical Figures Referenced
- **趙匡胤 (Zhao Kuangyin)**: Song Taizu, appears as "Brother Zhao"
- **趙光義 (Zhao Guangyi)**: Song Taizong, Kaifeng Prefect
- **李煜 (Li Yu)**: Last Southern Tang Emperor as "Nanzhu Prince"
- **柴榮 (Chai Rong)**: Zhou Shizong, mentioned in Suspended Sword lore
